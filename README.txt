Web Scraping de Notícias com Python, Qt Designer e Envio Programado por E-mail
Visão Geral

Este projeto oferece uma aplicação de web scraping em Python, combinada com a facilidade de design gráfico do Qt Designer, para coletar e visualizar notícias da web. Além disso, inclui a funcionalidade de enviar notícias programadas por e-mail aos usuários.
Funcionalidades Principais


1. Web Scraping com Python:
Utilizando a linguagem Python e a biblioteca BeautifulSoup, o projeto realiza web scraping para extrair informações relevantes de sites de notícias. Isso proporciona uma maneira eficiente de obter dados atualizados.


2. Interface Gráfica Intuitiva:
O Qt Designer é integrado para criar uma interface gráfica de usuário intuitiva. As janelas, botões e campos de entrada são projetados visualmente para uma experiência do usuário amigável.


3. Coleta e Visualização de Notícias:
Os usuários podem inserir palavras-chave ou selecionar categorias específicas de notícias. A aplicação utiliza técnicas de scraping para coletar dados relevantes, apresentando as notícias de forma organizada e legível.


4. Envio Programado por E-mail:
Uma funcionalidade permite aos usuários programar o envio de notícias para seus e-mails em horários específicos. Isso é alcançado por meio de bibliotecas como smtplib para envio de e-mails e schedule para agendar tarefas.


5. Persistência de Dados:
As preferências do usuário e as configurações de scraping são armazenadas para otimizar a experiência em execuções subsequentes.
6. Segurança e Ética

O projeto adere a práticas éticas e legais, respeitando os termos de serviço dos sites. Medidas de segurança são implementadas para garantir a manipulação segura de dados do usuário.
